# Text Cleaner

A fast, browser-based tool to clean messy text with precision. Strip out extra spaces, repair formatting, and prepare pasted content for notes, forms, and data work—all without leaving your browser.

## Features

- **Remove Extra Spaces** - Collapse repeated spaces and tabs into single spaces
- **Trim Line Edges** - Remove leading and trailing whitespace from each line
- **Normalize Blank Lines** - Reduce multiple consecutive blank lines to one
- **Remove Blank Lines** - Completely eliminate blank lines if needed
- **Fix Line Breaks** - Normalize mixed line breaks across your text
- **Real-time Stats** - Track characters saved and output size
- **Privacy-First** - Your content never leaves the page
- **One-Click Copy** - Export cleaned text instantly

## Getting Started

### Usage

1. Open `index.html` in your browser
2. Paste your messy text into the input area
3. Select which cleanup rules to apply using the checkboxes
4. Click **Clean Text** to process your content
5. Click **Copy Output** to copy the cleaned text to your clipboard

### Sample Data

Click **Inject Sample Text** to load example data and see the cleaner in action.

## Cleanup Options

- **Trim spaces at the start and end of each line** - Removes leading/trailing whitespace
- **Collapse repeated spaces and tabs into a single space** - Converts multiple whitespace characters to one
- **Reduce multiple blank lines to one blank line** - Compacts excessive vertical spacing
- **Remove all blank lines completely** - Strips all empty lines from the output
- **Normalize mixed line breaks** - Converts all line endings to consistent `\n` format
